# CINEMATICUM Real Case Authority Intake Docket

This object declares the real Case 001 authority intake docket after `OPEN_REAL_CASE_AUTHORITY_INTAKE`.

It is not a `NON_*` seal.
It is not a negative capability artifact.
It does not satisfy authority.
It does not advance state.
It does not issue a motion picture.
It does not admit media.
It does not create a release candidate.
It does not reopen the current snapshot.
It does not create a new snapshot.

It only declares the eight real-case authority slots that future explicit request objects may target:

1. DIRECTOR_AUTHORITY
2. FINAL_CUT_JURISDICTION
3. TIMELINE_EVIDENCE
4. RELEASE_ADMISSIBILITY
5. AUDIENCE_ARTIFACT
6. PROOF_ARTIFACT
7. OUTSIDER_REPLAY
8. TERMINAL_CLOSURE

The first allowed request object is:

```text
CASE_001_DIRECTOR_AUTHORITY_ADMISSION_REQUEST
````

Live authority request count remains zero in this object.
