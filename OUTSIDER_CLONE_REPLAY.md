# CINEMATICUM — Outsider Clone Replay

This is the public clone-time replay harness.

It defines what an outsider can do from a fresh checkout without private narration, private files, media payloads, model weights, paid APIs, or cloud render dependency.

## Fresh public checkout

    git clone https://github.com/kaaffilm/CINEMATICUM.git
    cd CINEMATICUM
    bash scripts/verify-all.sh

## Local harness

    bash scripts/verify-outsider-clone-replay.sh

## Boundary

Clone replay verifies the public repository perimeter.

Clone replay does not issue a film.

Clone replay does not make the case release-candidate-ready.

Clone replay does not admit media.

Clone replay does not execute film replay.

Clone replay does not create an admissibility verdict.

Clone replay does not create terminal closure.
